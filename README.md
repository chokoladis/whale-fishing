## Crypto Whale & Alpha Hunter Tracker.
Это платформа для мониторинга «умных денег» (Smart Money) и кошельков крупных игроков (китов). Пользователь выбирать по каким монетам отслеживать, платформа подбирает топ кошельки (юзер может сам закинуть список адресов, а система в реальном времени трекает их транзакции, считает профит и шлет алерты.
<br>
<br>
<b>Техническое ТЗ:</b>
1. Интеграция с Blockchain (The Data Source): Тебе не нужно поднимать свою ноду (это дорого). Для бесплатного старта используешь:
* Alchemy или Infura (Free Tier): Дают бесплатные лимиты на RPC-запросы к Ethereum/Polygon/L2.
* Etherscan/BscScan API: Для получения истории транзакций.
* DEX Screener / GeckoTerminal API: Чтобы подтягивать актуальные цены токенов-щиткоинов, которых нет на Coingecko.
2. Компонент "Watcher" (Backend Engine): Здесь ты разворачиваешь всю мощь Symfony:
* Messenger + Redis: Ты не можешь опрашивать блокчейн в основном потоке. Создаешь TransactionWatcherHandler.
* Scheduler (Symfony 6.3+): Настраиваешь периодический опрос пула «подозрительных» адресов.
* HttpClient: Настраиваешь его так, чтобы обходить лимиты (rate limiting) бесплатных API (через очереди и задержки).
3. Логика "Profit & Loss" (P&L) Calculator: Самая сложная инженерная часть.
* Тебе нужно не просто увидеть транзакцию, а посчитать: за сколько купил, за сколько продал, какая нереализованная прибыль.
* Challenge: Учет газа (комиссий) в стоимости позиции. Это покажет твою дотошность в расчетах.
Система Алертов (The Value):
* Telegram Notifier: Когда "кит" закупает новый токен на сумму > X, пользователю летит сообщение с кнопкой-ссылкой на контракт.
* Strategy Pattern: Реализуй разные типы алертов (на покупку, на продажу, на перевод на биржу — "Dump warning").


* DB: PostgreSQL (для истории) + Redis (для очередей и кэша цен).
* API: Alchemy (Free tier) + DEX Screener (Free).
* Deployment: Какой-нибудь Oracle Cloud Free Tier или минимальный VPS за $5.

Конкретный функционал для реализации (MVP):
1. Dashboard: Список отслеживаемых кошельков с их текущим балансом в USD.
2. Feed: Лента последних транзакций этих кошельков с пометками: BUY, SELL, TRANSFER.
3. Smart Filter: Возможность скрыть "пылевые" транзакции (меньше $100).
4. Telegram Bot: Команда /add 0x... для добавления кошелька в трекер прямо из телеги.
